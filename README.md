# ClariSync - SaaS Service Concept Site

中小企業・スタートアップ向けの業務管理クラウド「ClariSync（クラリシンク）」を想定した、モダンなコーポレートサイトの実装サンプルです。

##  URL
[https://clarisync.ymizomasa.com/](https://clarisync.ymizomasa.com/)

##  制作時間
- デザイン（Figma）: 25時間
- フロントエンド（HTML/CSS/JS）: 18時間
- WordPress構築: 7時間

##  概要
「企業の信頼性」と「先進性」の両立をテーマに、架空のSaaSスタートアップのブランディングを意識して制作しました。トップページから採用情報まで一貫したトーンで設計し、実務に近いサイト全体の導線構築を行っています。

##  デザインコンセプト：「信頼と先進性の融合」
- 配色**: 誠実さを表すブルー（#009BC6）と洗練されたグレー（#DDDDDD）を採用し、テック感のあるクールな印象を演出。
- タイポグラフィ: 「Source Sans 3」と「Noto Sans JP」を組み合わせ、スタイリッシュさと可読性を両立。
- レイアウト: 広めの余白と滑らかなスクロールアニメーションにより、次世代SaaSらしい「軽やかさ」と「心地よさ」を体現。

##  使用スキル・技術
### Core Skills
`HTML` / `CSS` / `JavaScript` / `WordPress` / `Figma`

### Technology Details
- Responsive Design: どのデバイスでも情報が整理されて表示されるよう、緻密なレスポンシブレイアウトを実装。
- Interaction: `Intersection Observer API` を用い、スクロールに連動した自然なフェードイン・スライドアニメーションを実装。
- WordPress (Original Theme):
  - 会社情報、News、採用情報などをCMS化。
  - 専門知識のない担当者でも、管理画面から容易にコンテンツ更新ができるよう設計。
- Figma Architecture: ワイヤーフレームからデザインモックまで作成。配色・コンポーネントのルールを統一し、実装時の整合性を高めています。

##  成果と学び
情報量の多い企業サイトを「シンプルかつストーリー性を持って伝える」ためのレイアウト設計を追求しました。トップから採用、問い合わせに至るまでのユーザー導線を意識したことで、単なるコーディングに留まらない「設計力」の向上を実感しています。
